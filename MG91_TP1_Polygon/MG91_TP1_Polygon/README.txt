README.txt
==========

Le repertoire contient des fichiers pour un TP sous Scilab
Pour lancer Scilab dans le repertoire il suffit de cliquer sur
un fichier *.sci
Ou sous linux de taper "scilex" ou "scilab" sous le repertoire
puis d'ouvrir le fichier dans l'éditeur de Scilab.

Le repertoire contient :
Le sujet : TPx.sci
Des scripts : a charger avec exec("nomfichier.sci",-1) 
data : un repertoire contenant des donnees : 
    Les fichiers sous data/*.txt a charger avec M=fscanfMat("nomfichier.txt")
figures : un repertoire contenant des figures : 
    Les fichiers sous figures/*.scg a charger avec xload("nomfichier.scg",numfig)
